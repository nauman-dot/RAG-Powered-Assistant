# 🔒 RAG Cybersecurity Knowledge Assistant

**A Retrieval-Augmented Generation (RAG) chatbot that answers questions using your own research paper (Network_Attack_CICDDos2019)**

Built as a final-year BS Computer Science project to demonstrate practical RAG application in **Network Security & Intrusion Detection**.

### ✨ Features
- Upload any cybersecurity PDF (your CICDDos2019 paper, journal articles, etc.)
- Instant, accurate answers grounded in the documents (zero hallucinations)
- Professional responses with bullet points, metrics, and source citations
- Built with modern 2026 tech stack
- Fully functional on Kaggle, Colab, and Hugging Face Spaces

### 🎯 Why This Project?
Turns my submitted research paper ("Network_Attack_CICDDos2019" – Journal of Network and System Management) into an interactive AI assistant. Recruiters and professors can now ask it anything about my feature engineering, models, or DDoS detection techniques.

### 🛠 Tech Stack
- **LLM**: Groq (Llama-3.1-8B-Instant) – blazing fast
- **Framework**: LangChain + ChromaDB (Vector Database)
- **Embeddings**: sentence-transformers/all-MiniLM-L6-v2
- **Frontend**: Gradio
- **PDF Processing**: PyPDFLoader + RecursiveCharacterTextSplitter
- **Certifications Applied**: Retrieval Augmented Generation (RAG), Vector Database Fundamentals, Generative AI with LLMs


### 📥 How to Run Locally / Kaggle
1. Add your `GROQ_API_KEY` in Kaggle Secrets
2. Upload your research PDF
3. Click "Build Knowledge Base"
4. Start asking questions!

### 📊 Results
- 100% context-grounded answers
- Works perfectly with my CICDDos2019 paper
- Ready for extension (voice input, agentic tools, multi-modal)
